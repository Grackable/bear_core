The Guide Settings build step loads guide settings from the asset defined in From Asset in the lower section of the Build tab. This allows reusing settings and locations of guides across assets independantly of the component settings.

Definition: Component settings are loaded from the guide.

Pivot Positions: Aligns Guide Pivot positions.

Pivot Orientations: Aligns Guide Pivot orientations.

Shape Settings: Loads Guide Shape settings.

Shape Transforms: Loads Guide Shape transforms.

Choose between All Components and Selected Components at the bottom of the Build tab.

Loading guide settings requires a guide file to be saved under File -> Save Guide.
