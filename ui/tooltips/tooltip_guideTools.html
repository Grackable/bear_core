There are two types of guide settings menus in BEAR:

- Component settings
- Guide Shape settings

If a component group (guide or rig) is selected, clicking Settings opens the Component settings. If a guide pivot or guide shape node is selected, the same button opens the Guide Shape settings.

The buttons Select Component, Select Pivot and Select Shape serve as a selection conversion for a more convenient workflow. With any node selected, these buttons help switching to the desired node type, making your work simpler and more efficient.

Multiple selections are supported.
