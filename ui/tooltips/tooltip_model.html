The Model build step imports the model and replaces the existing.

This build step is optional since it depends on your workflow or pipeline requirements. Using this build step requires either the model file to be located in the BEAR asset folder or a custom script for the model build step.

If you wish to move your model file to the BEAR asset folder, open your model in Maya and save it using the Save Model option in the File menu.

A Model Script can be assigned under Global Scripts in the BEAR Builder Settings. This custom script would handle the model import from any location specified by your pipeline.
