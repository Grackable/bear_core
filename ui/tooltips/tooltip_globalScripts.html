Global Scripts allows you to execute custom python scripts by build step.

Global Scripts Folder: The folder where all global scripts for your project are located.

Model Script: The script file for the Model build step.

Guide Script: The script file for the Guide build step.

Rig Script: The script file for the Rig build step.

Post Deform Script: The script file for the Post Deform build step.

Delivery Script: The script file for the Save Delivery option in the File menu.

Enable Script Execution on Build: Executes the scripts automatically on each build step. If unchecked, you can execute them using Execute Script for enabled build steps in the Edit menu.

Keep in mind you can also use custom scripts per asset by creating a Script component.
