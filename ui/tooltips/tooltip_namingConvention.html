Token Order: Define the token order. Every node's name of the rig will be assembled using the token order. You can also change the token order after the rig was built by clicking Apply Token Order.

Guide Root: The name guide parent group.

Rig Root: The name of the rig parent group.

Geometry Group: The name of your geometry group which contains all geometries.

Guide Pivot Token: The node type token for guide pivot nodes.

Guide Control Token: The node type token for guide control nodes.

Offset Token: The node type token for offset nodes.

Control Token: The node type token for control nodes.

Skin Joint Token: The node type token for skin joints.

Left Side Token: The side token for left side nodes.

Right Side Token: The side token for right side nodes.

Rename Objects will update the names of all guide and rig nodes in the scene.
